类型擦除

Java虚拟机中没有泛型对象，所有的对象都属于普通类。无论何时定义一个泛型类型，在Java虚拟机
中都会擦除类型变量，并替换为限定类型（无限定的变量用Object）

产生两个仅返回值类型不同的方法

Java泛型转换的总结

  • 虚拟机中没有泛型，只有普通的类和方法
  • 所有的类型参数都用它们的限定类型替代
  • 桥方法被合成保持多态
  • 为保持类型的安全性，必要时插入强制类型转换
